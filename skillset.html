Web Language I know...

HTML

Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

CSS

CSS stands for Cascading Style Sheets with an emphasis placed on “Style.”(defining things like headlines and paragraphs, and allowing you to embed images, video, and other media).

JavaScript

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

Word Press

WordPress is the world’s most popular tool for creating websites. WordPress is capable of creating any style of website, from a simple blog to a full-featured business website.

BootStrap

Quickly design and customize responsive mobile-first sites with Bootstrap, the world’s most popular front-end open source toolkit.

PHP

PHP is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.
